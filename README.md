## Testing for DSA problem.
### Structure of dir
```
problem_no/
├── generate.py
├── input
│   └── sample.txt (input files)
├── output
│   └── sample.txt (output files)
├── bin
│   └── your binary file
└── test.py
```
- Prerequisite: an execution file in bin.
- Usage:
```
  cd problem_no
  python3 test.py
```
